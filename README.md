# No Spoilers Chrome Extension
A chrome browser extension that will hide spoilers from showing up when you are binging a show. It will remove the Episode Image, Title, and Description.

    <img align="center" src="https://github.com/WilliamPasternak/No-Spoilers-Chrome-Extension/blob/main/No%20Spoilers.gif" alt="Video showing functionality of 'No Spoilers' chrome extension">
</p>
 
## Services Currently Supported ##
* Hulu

## How It's Made:

**Tech used:** Javascript

## How to Install:
* Clone or download this repository.
* Enable developer mode in your chrome browser at chrome://extensions.
* Click "Load unpacked" and point to the root directory of the extension.
* Click 'Details" and change 'Site Access' to 'On-Click'

## How to Use:
* Navigate to a series' episode list on your streaming service
* Click the No Spoilers extension.

## Optimizations
* Add support for additional streaming platforms.
* Add 'always on' support. In the current version, the extension must be clicked to be used.
